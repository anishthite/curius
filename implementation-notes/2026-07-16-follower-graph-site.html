Algorithm

  1. Read all users plus follower/following counts from SQLite.
  2. Embed compact {nodes, edges} JSON in the static page.
  3. Default to top users; on search/click render the selected user's one-hop neighborhood.

UI layout

  1. Toolbar: search, minimum followers filter, graph mode, reset.
  2. Graph: SVG force-ish layout, clickable nodes, arrows for follows.
  3. Sidebar: selected profile counts plus follower/following lists.
