Algorithm

  1. Read supported text files under --index.
  2. Split content on blank-line paragraphs.
  3. Tokenize with [a-z0-9']+.
  4. Rank chunks by |query_terms ∩ chunk_terms| / sqrt(|chunk_terms|).
  5. Print top snippets with source paths, or I don't know from the index..
